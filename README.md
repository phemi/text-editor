A Simple Text Editor (Notepad love!).

![alt text](app.png)


## Features
    *Create, Read, Edit, Save text file
    *Search functionalities
    *Color Change
    *Read text files in audio (in progress)
    

### Requirements
    *Java: JDK >= 7

<br><br>

### How to install <br>
    *Clone Repo <br>
    *Run Main.java class <br>
